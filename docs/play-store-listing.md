## Full Description (max 4000 characters)

```
Amiberry is a fast, accurate Amiga emulator built on the WinUAE core. It brings the classic Amiga experience to your Android device with optimized performance and an intuitive touch interface.

EMULATED SYSTEMS
- Amiga 500 / 500+ (OCS/ECS)
- Amiga 1200 (AGA)
- Amiga 4000
- Amiga CD32
- And more configurations via custom setup

KEY FEATURES

Accurate Emulation
Full Amiga hardware emulation including custom chips (Agnus, Denise, Paula), 68000 through 68060 CPUs, and AGA chipset support. Run your favorite games, demos, and productivity software just like on real hardware.

WHDLoad Integration
Built-in WHDLoad booter lets you launch hard-drive installed games directly. Point Amiberry at your WHDLoad collection and start playing — all configuration is handled automatically.

Touch Controls
Purpose-built arcade-style on-screen joystick with a movable ball-top knob and dome fire buttons. Slide your finger to move, tap to fire. Supports multi-touch for simultaneous movement and shooting.

Controller Support
Full support for Bluetooth and USB gamepads. Compatible with standard Android controllers, RetroArch-mapped controllers, and keyboard/mouse input.

Storage Formats
Supports a wide range of Amiga disk and file formats:
- Floppy disks: ADF, IPF, DMS, FDI, EXE
- Hard drives: HDF, directory hard drives
- CD-ROM: ISO, CUE/BIN (for CD32 titles)
- Archives: LHA, ZIP, 7Z (with auto-extraction)
- Compressed disk images: CHD (via zstd)

JIT Compiler
High-performance Just-In-Time compiler provides smooth emulation even for demanding AGA software and 68040/68060 configurations on supported devices.

Display Options
Multiple scaling and rendering options including CRT shader effects for authentic retro visuals. Supports both landscape and portrait orientations.

Save States
Save and restore your emulation state at any time. Never lose progress.

Open Source
Amiberry is free and open-source software (GPLv3). The complete source code is available on GitHub.

IMPORTANT: No Amiga ROM files or software are included with this app. You must provide your own legally obtained Kickstart ROMs and Amiga software. Amiga Forever (https://www.amigaforever.com) is a legal source for Kickstart ROMs.

For documentation, guides, and support:
- Wiki: https://github.com/BlitterStudio/amiberry/wiki
- Discord: https://discord.gg/wWndKTGpGV
- Website: https://amiberry.com
```
